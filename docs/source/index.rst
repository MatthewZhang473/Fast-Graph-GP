Fast-Graph-GP documentation
===========================

Fast-Graph-GP provides scalable Gaussian process inference on graphs using
graph random features and sparse graph kernel approximations.

Installation
============

Install from PyPI:

.. code-block:: bash

   pip install fast-graph-gp
